ARQUITECTURA DEL NÚCLEO

Brain Engine

El bot utiliza Gemini 1.5 Flash para procesar prompts. Esta integración permite a Ø𝐓𝐈 analizar texto y generar respuestas coherentes con una personalidad definida.

Bot Logic

Construido sobre Discord.js v14. Maneja un sistema de comandos de barra optimizado para evitar el error de "Interaction Unknown" mediante el uso de deferReply.

Infrastructure

Ejecutado en un entorno de Node.js dentro de Replit. Incluye un servidor Express que mantiene la instancia activa y sirve los logs en tiempo real para la web.

Security

Implementación de Variables de Entorno para la gestión de API Keys. Esto asegura que los tokens sensibles jamás se filtren en el repositorio de GitHub.
